Knight Runner
Knight Runner é um jogo desenvolvido pelos alunos Guilherme Kaidei e Henrique Mayor como um projeto do curso de Ciência da Computação do Insper
Descrição do jogo:
Knight Runner é um jogo em 2D em que um cavaleiro corre enquanto desvia de obstáculos e enfrenta inimigos. O objetivo do jogo é sobreviver o máximo de tempo e matar o maior número de inimigos possíveis para obter pontos. Enquanto pula e atira para vencer desafios, o jogador deve levar em consideração as 3 vidas que possui. Se o cavaleiro perder todas as vidas, o jogo acaba.

Como jogar:

Para jogar, é necessário ter o Python 3 instalado na máquina. Além disso, você deve instalar a biblioteca pygame com o comando: pip install pygame. Após instalar a biblioteca, utilize o botão abaixo para baixar o jogo e extraia o arquivo. Após isso, execute o arquivo jogo.py e o jogo será aberto em uma nova janela.
